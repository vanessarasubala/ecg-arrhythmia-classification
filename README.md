# ECG Arrhythmia Classification

This project aims to build a medical AI system to classify ECG (electrocardiogram) signals into normal and abnormal heart rhythms.

## Objective
- Understand ECG signal data
- Perform preprocessing on labeled medical data
- Train a baseline machine learning model
- Evaluate performance using classification metrics

## Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

Project initialization completed. Next step: dataset exploration.
